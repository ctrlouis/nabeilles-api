# nabeilles api


## Get started

### First build
**Create/rebuild containers:**
```
sudo docker-compose up --build
```

### Run project
**Start containers:**
```
sudo docker-compose up --build
```

**Usefull url :**
- Request `http://localhost:3000` for connect to the api.
- phpmyadmin : `http://localhost:8080`
- adminer : `http://localhost:8081`
